# Cyclic Groups
**Formal definition of cyclic groups**:

A cyclic group is a type of algebraic structure that exhibits a particular pattern of element generation through repeated application of a single generator element. It can be formally defined as follows:

**Definition**: Let G be a group, and let 'a' be an element in G. If there exists an integer 'n' such that:

1. $a^n = e$, where 'e' is the identity element of G, and
2. For every integer 'k', there exists an element $a^k$ in G,

then G is said to be a cyclic group generated by 'a', and 'a' is called the generator of G.

In this definition:

- $a^n = e$ implies that 'a' raised to the power of 'n' results in the identity element 'e' of the group.
- The second condition ensures that every element of the group can be expressed as a power of 'a'. In other words, the repeated application of 'a' and its inverses generates all elements of the group.

Cyclic groups are often denoted as $\langle a \rangle$, where 'a' is the generator element. These groups can be finite or infinite, depending on the value of 'n'. If 'n' is finite, the cyclic group is finite; if 'n' is infinite, the cyclic group is infinite.

Additionally, cyclic groups have some important properties:
- A cyclic group is always abelian (commutative), meaning that the order of multiplication does not matter.
- The order of a cyclic group is equal to the order of its generator. If 'n' is finite, the order of the group is 'n'; if 'n' is infinite, the group is also infinite.
- Every subgroup of a cyclic group is cyclic.

Cyclic groups provide a fundamental example of algebraic structures and play a crucial role in group theory. They are used to study various properties of groups and serve as building blocks for understanding more complex group structures.
# Infinite Cyclic Groups:

An infinite cyclic group is a specific type of group that is generated by a single element and contains infinitely many elements. Let's formally define an infinite cyclic group:

**Definition**: An infinite cyclic group, denoted as $\langle a \rangle$, is a group generated by a single element 'a' such that:
1. The set of elements in the group is given by $\langle a \rangle = \{a^n \mid n \in \mathbb{Z}\}$, where $\mathbb{Z}$ represents the set of all integers.
2. The group operation is defined as multiplication of powers of 'a', i.e., for any two elements $a^n$ and $a^m$ in the group, their product is $a^n * a^m = a^{n + m}$.

In this definition:

- The elements of the group are all integer powers of 'a'. This set is infinite because it includes both positive and negative powers of 'a', resulting in an uncountably infinite number of elements.

- The group operation is consistent with the idea of combining powers of 'a' through exponent addition, which is a consequence of the group's definition.

- The generator 'a' can be any element of the group, but it generates the entire group by exponentiation. It's worth noting that there can be multiple infinite cyclic groups, each generated by a different element.

**Properties of Infinite Cyclic Groups**:

1. **Abelian Group**: Infinite cyclic groups are always abelian (commutative), meaning that the order of multiplication does not matter. This property follows directly from the commutativity of integer addition.

2. **Order**: The order of an infinite cyclic group is infinite since it contains infinitely many elements. It is denoted as $|\langle a \rangle| = \infty$.

3. **Subgroups**: Any subgroup of an infinite cyclic group is also an infinite cyclic group. It is generated by a power of the original generator 'a'.

Infinite cyclic groups are fundamental in abstract algebra and group theory. They provide a simple yet illustrative example of an infinite group and serve as building blocks for more complex group structures.

# **Theorem**: Any number from 0 to n - 1 that is relatively prime to n will generate the group $\mathbb{Z}_n$ under addition modulo n.

**Proof**:

Let's consider the group $\mathbb{Z}_n$, which consists of integers modulo n, i.e., $\mathbb{Z}_n = \{0, 1, 2, \ldots, n - 1\}$. This group is closed under addition modulo n, and we want to show that any integer 'a' relatively prime to n will generate this entire group.

To prove this, we will use the fact that if 'a' and 'n' are relatively prime, there exists an integer 'x' such that:

$$
ax \equiv 1 \pmod{n}
$$

This is a consequence of the Bezout's identity, which states that if 'a' and 'n' are coprime, then there exist integers 'x' and 'y' such that:

$$
ax + ny = 1
$$

Now, let's consider the set of elements generated by 'a' modulo n:

$$
\{a, 2a, 3a, \ldots\}
$$

We want to show that every element in $\mathbb{Z}_n$ can be generated using this set. 

**Step 1: Prove Closure**

First, we need to show that the set is closed under addition modulo n. Let's take two elements, 'ka' and 'ja', where 'k' and 'j' are integers. When we add them modulo n:

$$
(ka + ja) \pmod{n} = (k + j)a \pmod{n}
$$

Since 'a' is relatively prime to 'n', we know that there exists an 'x' such that:

$$
ax \equiv 1 \pmod{n}
$$

Therefore, we can express '(k + j)a' as:

$$
(k + j)a \equiv kax + jax \equiv kx + jx \pmod{n}
$$

Since 'kx' and 'jx' are integers, the sum '(kx + jx)' is also an integer. Therefore, '(k + j)a' is in the set, and the set is closed under addition modulo 'n'.

**Step 2: Prove Generatorship**

Now, let's show that the set generates all elements in $\mathbb{Z}_n$. Take an arbitrary element 'm' from $\mathbb{Z}_n$. We need to show that there exists an integer 'k' such that:

$$
ka \equiv m \pmod{n}
$$

Since 'a' and 'n' are coprime, we can use the fact that 'ax' is equivalent to '1' modulo 'n' as established earlier. Therefore, we can multiply both sides of the congruence by 'm':

$$
ax \cdot m \equiv 1 \cdot m \pmod{n}
$$

This simplifies to:

$$
am \equiv m \pmod{n}
$$

Now, we have an equation that states 'am' is congruent to 'm' modulo 'n'. Since 'a' is relatively prime to 'n', 'am' will cover all residues modulo 'n' from 0 to n - 1. Therefore, 'm' can be expressed as '(am)' for some integer 'k', which means that 'm' is in the set generated by 'a'.

Thus, we have shown that any number 'a' relatively prime to 'n' generates the entire group $\mathbb{Z}_n$ under addition modulo 'n'. This concludes the proof.

# Orbits
**Orbit of a Group**:

In group theory, the concept of an "orbit" is often associated with the action of a group on a set. The orbit of a group under this action represents the set of all possible outcomes or elements that can be obtained by applying the group elements to a given element of the set. To formally define the orbit of a group, we need to introduce some key concepts and notations.

Let G be a group, and let S be a set on which G acts. The action of G on S is represented as a function:

$$
\cdot : G \times S \to S
$$

where for each g in G and s in S, g · s denotes the result of applying the group element g to the element s. This action satisfies the following properties:

1. **Identity Element**: For any element s in S, the identity element e of G leaves s unchanged, i.e., e · s = s for all s in S.

2. **Compatibility**: The action is compatible with the group operation in G, meaning that for all g, h in G and s in S, (g · h) · s = g · (h · s).

Now, we can define the orbit of an element s in S under the action of G:

**Definition**: The orbit of an element s in S under the action of G, denoted as $Orb_G(s)$, is the set of all elements in S that can be obtained by applying elements of G to s:

$$
Orb_G(s) = \{g \cdot s \mid g \in G\}
$$

In other words, $Orb_G(s)$ contains all the elements that can be reached from s by applying the group elements of G through the group action.

**Properties of Orbits**:

1. **Orbit Equivalence**: Two elements s and t in S are said to be in the same orbit under the action of G (denoted as s ~ t) if there exists an element g in G such that t = g · s. In other words, s ~ t if they belong to the same orbit.

2. **Orbit Partition**: The set S can be partitioned into disjoint orbits under the action of G. Each orbit is a subset of S containing elements that are related through the group action.

3. **Stabilizer Subgroup**: For each element s in S, the stabilizer subgroup of s, denoted as $Stab_G(s)$, is the subgroup of G consisting of elements that leave s unchanged, i.e., $Stab_G(s)$ = {g in G | g · s = s}.

Understanding orbits is crucial in group theory, especially when studying group actions, permutation groups, and applications in various mathematical and scientific contexts. Orbits help analyze how a group transforms a set and provide insights into the structural properties of groups and their actions.

# Order of an element as the size of its orbit
In group theory, the order of an element in a group is intimately connected to the size of its orbit under certain group actions. Let's formally define these concepts:

**Definition 1: Order of an Element (Group Theory)**
Given a group $G$ and an element $g$ in $G$, the order of $g$, denoted as $|g|$, is defined as the smallest positive integer $n$ such that $g^n = e$, where $e$ is the identity element of the group $G$. If there is no such positive integer $n$, the order of $g$ is considered infinite.

**Definition 2: Group Action**
Let $G$ be a group and $X$ be a set. A group action of $G$ on $X$ is a function $\cdot: G \times X \rightarrow X$ that satisfies the following properties for all $g, h \in G$ and $x \in X$:
1. $e \cdot x = x$, where $e$ is the identity element of $G$.
2. $(gh) \cdot x = g \cdot (h \cdot x)$, where $(gh)$ represents the group operation in $G$.

**Definition 3: Orbit**
Given a group $G$ acting on a set $X$ and an element $x \in X$, the orbit of $x$ under the action of $G$, denoted as $\text{Orb}_G(x)$, is defined as the set of all elements in $X$ that can be obtained by applying elements of $G$ to $x$:
$$\text{Orb}_G(x) = \{g \cdot x \mid g \in G\}.$$

**Theorem: Order of an Element and Size of Its Orbit**
For a group $G$ acting on a set $X$, if $g$ is an element of $G$ and $x$ is an element of $X$, then the order of $g$ is equal to the size (cardinality) of its orbit $\text{Orb}_G(x)$ under the action of $G$:
$$|g| = |\text{Orb}_G(x)|.$$

**Proof:**
Let $n = |g|$ be the order of $g$, i.e., $g^n = e$. Consider the orbit of $x$ under the action of $G$:
$$\text{Orb}_G(x) = \{g \cdot x, g^2 \cdot x, \ldots, g^n \cdot x\}.$$
Since $g^n = e$, we can simplify this to:
$$\text{Orb}_G(x) = \{g \cdot x, g^2 \cdot x, \ldots, e \cdot x\}.$$
By the definition of a group action, $e \cdot x = x$. Therefore, we have:
$$\text{Orb}_G(x) = \{g \cdot x, g^2 \cdot x, \ldots, x\}.$$
The size (cardinality) of this set is $n$, which is the order of $g$. Hence, $|g| = |\text{Orb}_G(x)|$.

This theorem establishes a fundamental connection between the order of an element in a group and the size of its orbit under a group action. It provides a useful tool for analyzing groups and their actions.

# Negative Exponents in Group Theory
In group theory, negative exponents play a significant role, particularly in the context of inverses and group operations. Let's delve into the concept of negative exponents in group theory.

**Definition: Negative Exponent in Group Theory**
Given a group $G$ with an operation denoted as $\cdot$, and an element $a$ in $G$, the negative exponent of $a$, denoted as $a^{-n}$ for a positive integer $n$, is defined as the inverse of $a$ raised to the power of $n$. Formally:
$$a^{-n} = (a^{-1})^n.$$
Here, $a^{-1}$ represents the inverse of $a$ in the group $G$, and the exponent $n$ signifies the number of times we apply the inverse operation.

**Theorem: Properties of Negative Exponents**
Let $G$ be a group, and $a$ and $b$ be elements of $G$. For any positive integer $n$, the following properties hold:

1. $(a^{-n})^{-1} = a^n$: The inverse of the negative exponent of $a$ is the positive exponent of $a$.
2. $(ab)^{-n} = b^{-n}a^{-n}$: The negative exponent distributes over the product of elements in the group.
3. $(a^n)^{-1} = a^{-n}$: The inverse of the positive exponent of $a$ is the negative exponent of $a$.

**Proof of Property 1:**
To prove that $(a^{-n})^{-1} = a^n$, consider the left-hand side:
$$(a^{-n})^{-1} = (a^{-1})^{(-n)^{-1}} = (a^{-1})^{-(-n)} = (a^{-1})^n.$$
Now, we have shown that the left-hand side is equal to $a^n$, which proves Property 1.

**Proof of Property 2:**
To prove that $(ab)^{-n} = b^{-n}a^{-n}$, consider the left-hand side:
$$(ab)^{-n} = ((ab)^{-1})^n = (b^{-1}a^{-1})^n = (b^{-1})^n(a^{-1})^n = b^{-n}a^{-n}.$$
Thus, we have shown that the left-hand side is equal to the right-hand side, which proves Property 2.

**Proof of Property 3:**
To prove that $(a^n)^{-1} = a^{-n}$, consider the left-hand side:
$$(a^n)^{-1} = (a^{-1})^{n^{-1}} = (a^{-1})^{-n} = a^{-n}.$$
So, the left-hand side is equal to the right-hand side, which proves Property 3.

Negative exponents in group theory allow us to express inverses and manipulate group elements effectively. They are a fundamental concept in understanding group operations and properties.

# Abelian Groups
An Abelian group, also known as a commutative group, is a fundamental concept in group theory. It refers to a set equipped with an operation that satisfies the group axioms, with the additional property that the operation is commutative. In other words, for any elements 'a' and 'b' in the group, the order in which you apply the operation doesn't matter; 'a * b' is equal to 'b * a'. 

**Group Axioms:**

1. **Closure Property:** For all elements 'a' and 'b' in the group, the result of the group operation, denoted as 'a * b', is also in the group.

2. **Associative Property:** For all elements 'a', 'b', and 'c' in the group, the operation is associative, i.e., (a * b) * c = a * (b * c).

3. **Identity Element:** There exists an identity element 'e' in the group such that for any element 'a' in the group, 'e * a = a * e = a'.

4. **Inverse Element:** For every element 'a' in the group, there exists an inverse element 'a⁻¹' such that 'a * a⁻¹ = a⁻¹ * a = e', where 'e' is the identity element.

**Commutative Property (Abelian Property):**

An Abelian group satisfies an additional property:

5. **Commutative Property:** For all elements 'a' and 'b' in the group, the group operation is commutative, i.e., 'a * b = b * a'.

**Examples of Abelian Groups:**

1. The integers (ℤ) under addition form an Abelian group because addition is both associative and commutative, and there exists an identity element (0) and inverses for every integer.

2. The set of real numbers (ℝ) excluding zero under multiplication forms an Abelian group because multiplication is both associative and commutative, and there exists an identity element (1) and inverses for every nonzero real number.

3. The set of integers modulo 'n' (ℤ/nℤ) under addition modulo 'n' forms an Abelian group for a positive integer 'n'.

4. The set of complex numbers (ℂ) excluding zero under multiplication is an Abelian group.

**Properties of Abelian Groups:**

1. Abelian groups are closed under the group operation.

2. Abelian groups are associative under the group operation.

3. Abelian groups have an identity element, and each element has an inverse.

4. Abelian groups satisfy the commutative property, meaning the order of the elements in the group operation doesn't affect the result.
